# 火把工具箱

一个简单易用的在线工具集合，提供多种图片处理功能。

## 功能特点

### 1. 图片切割工具
- 支持自定义切割尺寸
- 支持预览切割效果
- 支持批量下载切割后的图片

### 2. 图片压缩工具
- 支持多种压缩质量选择
- 支持多种输出格式（JPG、PNG、WebP）
- 支持批量处理多张图片
- 实时显示压缩效果和大小对比

### 3. 图片取色工具
- 支持提取图片主色调
- 自动生成颜色的明暗变体
- 支持一键复制颜色代码
- 支持批量处理多张图片
- 支持拖拽上传

## 使用方法

1. 直接访问工具网站
2. 选择需要使用的工具
3. 上传图片（支持拖拽或点击上传）
4. 根据需要调整参数
5. 获取处理结果

## 技术栈

- HTML5
- Tailwind CSS
- JavaScript
- Color Thief (颜色提取)
- Browser Image Compression (图片压缩)
